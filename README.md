# Targus Group Corporate Website

Static corporate website for Targus Nig Ltd and operating business websites under the Targus Group domain.

## Sites

- `www.targusgrp.com` - Targus Nig Ltd group website
- `www.tcop.targusgrp.com` - Targus Cocoa and Oil Palm Plantation
- `targuspro.targusgrp.com` - Targus Properties

## Deployment

This project is prepared for Vercel. The `vercel.json` file routes the business subdomains to their dedicated pages.

See `VERCEL-DEPLOYMENT.md` for deployment steps.
